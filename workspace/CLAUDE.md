# CLAUDE.md

このファイルはワークスペース内で作業する際の Claude Code (claude.ai/code) へのガイダンスを提供します。

## ワークスペース概要

`~/workspace` は開発リポジトリ・ノート・作業タスクを含むルートワークスペースディレクトリです。

## ディレクトリ構成

- `repos/` — 開発リポジトリ。詳細は `repos/CLAUDE.md` を参照。
- `notes/` — Obsidian等で管理する個人ノート。詳細は `notes/CLAUDE.md` を参照。
- `tasks/` — 作業スペース（タスクごとのサブディレクトリ）。詳細は `tasks/CLAUDE.md` を参照。
